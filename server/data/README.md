# 知识库文档目录

此目录用于存放Vickie的个人资料文档，这些文档将被上传到Gemini File Search Store，作为RAG系统的知识库。

## 支持的文件格式

- `.txt` - 纯文本文件
- `.pdf` - PDF文档
- `.docx` / `.doc` - Word文档
- `.md` - Markdown文件
- `.json` - JSON文件

## 使用方法

1. 将您的文档放入此目录
2. 运行 `npm run setup-store` 初始化File Search Store
3. 脚本会自动上传所有文档并配置环境变量

## 建议的文档内容

- 个人简历/CV
- 项目经历详情
- 技能清单
- 工作经验描述
- 个人简介
